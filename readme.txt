ارسال کالاهای فروشگاه شما به سایت دکمه

== Description ==
دکمه یک شبکه اجتماعی برای خرید از فروشگاه‌های اینترنتی است. در دکمه می توانید نظرات خود را درباره کالاها به اشتراک بگذارید و از تخصص دیگران برای خرید استفاده کنید.
= امکانات =x
 * پنل تنظیمات ساده و کاربرپسند.
 * قابلیت ارسال کالا به صورت تکی.
 *قابلیت ارسال کالا به صورت گروهی.
 *قابلیت ارسال کالاهای دسته انتخاب شده توسط کاربر.

== Installation ==
1. پوشه `dokme` را در مسیر `/wp-content/plugins/` آپلود کنید.

2. افزونه را از طریق منوی 'افزونه ها' در وردپرس فعال کنید.

3. تنظیمات افزونه را می توانید از منو وردپرس انجام دهید.
